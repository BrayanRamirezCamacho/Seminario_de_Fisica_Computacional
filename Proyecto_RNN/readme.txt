Proyecto red neuronal RNN:
Predicicción de los precios de la bolsa usando redes LSTM con Keras
El objetivo de este proyecto es entrenar una red neuronal tipo RNN (Recursive Neural Network) con la finalidad de hacer predicciones en las tendencias de la bolsa de valores en base a datos ordenados de manera secuencial (series temporales). Se utilizará la librería Keras para construir la red neuronal.
